Project Overview

I use human-centered methodologies as a creative approach to problem-solving.

Kick-Off Workshop
Two day workshop

Collaborative session to uncover user needs, business viability, and technical feasibility.

User Flow
The path a user follows through an application

It helps to think through the design before a feature is actually developed. It allow to interject the user into the flow of the application and determine if the conceptual model agrees with the user model.

Experience Map
Visual representation of the overall experience

It tells the story of the customer's experience: from initial contact, through the process of engagement and into a long-term relationship. It may focus on a particular part of the story or give an overview of the entire experience.
